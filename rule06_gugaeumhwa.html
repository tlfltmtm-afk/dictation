2부: 소리들의 만남과 변화

✨ 6. '이' 마법 (구개음화)

✨ [받아쓰기 나라의 신비한 마법사!]

"우리 같이 놀자!" 소리는 [가치]! '같이'의 받침 'ㅌ'(티읕)이 'ㅊ' 소리가 나네요!

"해돋이 보러 가자!" 소리는 [해도지]! 'ㄷ'(디귿)이 'ㅈ' 소리가 났고요!

딱딱한 'ㄷ', 'ㅌ'이 'ㅈ', 'ㅊ'으로 변신하는 신비한 마법! 이름하여... ['이' 마법]!

📖 [규칙 이야기: "치카치카 이 닦자!" 부드럽게 변신!]

딱딱하고 힘센 **'ㄷ'**과 **'ㅌ'** 받침이 **'이' 마법사** (ㅣ, ㅑ, ㅕ, ㅛ, ㅠ)를 만나면 놀라운 일이 벌어져요!

'이' 마법사는 이 기사님들을 **부드러운 소리**로 바꿔준답니다.

  • 'ㄷ' 받침 + '이' 마법사 → 부드러운 'ㅈ' 소리로 변신! (예: 굳 + 이 → [구지])
  • 'ㅌ' 받침 + '이' 마법사 → 부드러운 'ㅊ' 소리로 변신! (예: 같 + 이 → [가치])

핵심: 소리는 마법에 걸려 [가치]로 변해도, 글씨를 쓸 때는 마법에 걸리기 전의 **원래 모습 '같이'**를 써 줘야 해요!

🔍 ['이' 마법 소리를 확인해요! (예시)]

  • 같이 (ㅌ + 이) → 'ㅌ'이 '이'를 만나 'ㅊ'으로 변신! → 소리는 **[가치]**
  • 굳이 (ㄷ + 이) → 'ㄷ'이 '이'를 만나 'ㅈ'으로 변신! → 소리는 **[구지]**
  • 해돋이 (ㄷ + 이) → 'ㄷ'이 '이'를 만나 'ㅈ'으로 변신! → 소리는 **[해도지]**
  • 붙여 (ㅌ + 여) → 'ㅌ'이 '여'를 만나 'ㅊ'으로 변신! → 소리는 **[부처]**

✍️ [나도 할 수 있어! (연습 문제)]

'이' 마법사를 만났어요! 어떻게 변신할까요? 빈칸에 소리 나는 대로 적어보세요.

  1. 맏 + 이 (첫째라는 뜻!) → [마지]
  2. 땀받 + 이 → [땀바지]
  3. 밭이 (ㅌ + 이) → [바치]

⚠️ [잠깐! '이사 가는 받침'(1번)과 절대 헷갈리지 마!]

이 마법은 **꼭! '이' 마법사('ㅣ'나 'ㅑ, ㅕ, ㅛ, ㅠ')**를 만났을 때만 일어나요!

만약 '아, 어, 으' 같은 다른 친구를 만나면? 마법은 일어나지 않고, **그냥 '이사'를 가요! (1번 규칙)**

밭 + 이 → [바치] (O) (마법 발동!) / 밭 + 을 → [바틀] (O) ('을'은 마법사가 아니에요! 1번 규칙으로 이사!)

📌 [핵심 요약! (기억 상자)]

[ㄷ, ㅌ] 받침 + ['이' 마법사] → "부드러워져라! 얍!" 하고 [ㅈ, ㅊ] 소리로 변신! 소리는 [가치]로 변신해도, 글씨는 마법 전 '같이' 쓰기!
