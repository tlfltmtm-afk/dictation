2부: 소리들의 만남과 변화

💪 4. 힘센 소리 (된소리되기)

💪 [받아쓰기 나라의 파워 업!]

"국밥 먹고 학교 갈 거예요!" 소리는 [국빱]! [학꾜]!

'밥'의 'ㅂ'이 갑자기 [ㅃ] 하고 힘이 세졌어요! '교'의 'ㄱ'도 [꾜] 하고 단단해졌네요. 왜일까요?

이건 바로 앞에 있는 **'받침 친구'**를 만났기 때문이에요! 이름하여... [힘센 소리] 변신!

📖 [규칙 이야기: "얍!" 하고 힘을 내요!]

착하고 부드러운 예사소리 친구들 **'ㄱ, ㄷ, ㅂ, ㅅ, ㅈ'**이 길을 가고 있었어요.

그런데 'ㄱ, ㄷ, ㅂ'처럼 힘세 보이는 **받침 친구(받침 대장)**를 딱! 마주치면, 깜짝 놀라서 "얍!" 하고 같이 힘을 줘버려요! → **[ㄲ, ㄸ, ㅃ, ㅆ, ㅉ] (된소리)**로 변신!

  • 'ㄱ' 받침을 만나면 → 뒤의 'ㄱ'이 **[ㄲ]**으로 변신! (예: 학 + 교 → [학꾜])
  • 'ㄷ' 받침을 만나면 → 뒤의 'ㄷ'이 **[ㄸ]**으로 변신! (예: 깍 + 두기 → [깍뚜기])
  • 'ㅂ' 받침을 만나면 → 뒤의 'ㅂ'이 **[ㅃ]**으로 변신! (예: 국 + 밥 → [국빱])
  • **주의!** 'ㅅ', 'ㅈ'도 힘이 세져서 [ㅆ], [ㅉ]이 돼요! (예: 밥상 → [밥쌍], 옆집 → [엽찝])

핵심: 소리는 힘이 세져도, 글씨는 변신하기 전의 원래 모습(학교)으로 써 줘야 해요!

🔍 [힘센 소리를 확인해요! (예시)]

  • 학교 (ㄱ 받침 + ㄱ) → 소리는 **[학꾜]**
  • 깍두기 (ㄱ 받침 + ㄷ) → 소리는 **[깍뚜기]**
  • 옷고름 (ㄷ 받침 + ㄱ) → 소리는 **[옫꼬름]** (옷의 'ㅅ'은 'ㄷ' 대장 소리!)
  • 옆집 (ㅂ 받침 + ㅈ) → 소리는 **[엽찝]** (옆의 'ㅍ'은 'ㅂ' 대장 소리!)

✍️ [나도 할 수 있어! (연습 문제)]

힘센 소리로 변신시켜 볼까요? 빈칸에 소리 나는 대로 적어보세요.

  1. 책상 → [책쌍]
  2. 부엌 + 도 → [부억또] (부엌은 2번 규칙 적용 후!)
  3. 입 + 다 → [입따]

⚠️ [잠깐! '이사 가는 받침'(1번)과 헷갈리면 안 돼요!]

이 '힘센 소리' 규칙은 뒤에 오는 글자가 **힘이 있는 자음**일 때만 생겨요!

만약 뒤에 'ㅇ'으로 시작하는 **'빈방'**이 오면? (1번 규칙) 힘을 줄 필요 없이 "빈방이다!" 하고 그냥 **이사를 가요!**

부엌 + 도 → [부억또] (O) (자음이 와서 힘이 세짐) / 부엌 + 이 → [부어키] (O) (빈방이 와서 이사 감)

📌 [핵심 요약! (기억 상자)]

[ㄱ, ㄷ, ㅂ] 받침 소리 + [ㄱ, ㄷ, ㅂ, ㅅ, ㅈ] 예사소리 → "얍!" 하고 [ㄲ, ㄸ, ㅃ, ㅆ, ㅉ] 힘센 소리로 변신! 소리는 [학꾜], 글씨는 '학교' 쓰기!
