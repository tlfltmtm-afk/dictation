2부: 소리들의 만남과 변화

🤖 7. 거센소리 로봇 (자음 축약)

🤖 [받아쓰기 나라의 합체 로봇!]

"생일 축하합니다!" 소리는 [추카]! '축'의 'ㄱ' 받침과 '하'의 'ㅎ'이 만났는데, 두 소리가 사라지고 갑자기 'ㅋ'(키읔) 소리가 튀어나왔어요!

두 글자가 하나로 합쳐져서 더 힘세고 바람 소리가 나는 로봇으로 변신한 거예요! 이름하여... [거센소리 로봇]!

📖 [규칙 이야기: "치익- 합체!" 파워 부스터 'ㅎ'!]

기본 로봇 **'ㄱ, ㄷ, ㅂ, ㅈ'**이 **'파워 부스터' 'ㅎ'(히읗)**을 만나면 "합체 신호다!"

"치익-!" 하고 둘이 하나로 합쳐져서 **'거센소리 로봇'**으로 변신해요! (순서는 상관없어요! 받침 'ㄱ,ㄷ,ㅂ,ㅈ' + 'ㅎ' 이든 'ㅎ' + 'ㄱ,ㄷ,ㅂ,ㅈ' 이든 같아요!)

  • ㄱ + ㅎ = ㅋ (예: 축 + 하 → [추카])
  • ㄷ + ㅎ = ㅌ (예: 맏 + 형 → [마텽])
  • ㅂ + ㅎ = ㅍ (예: 입 + 학 → [이팍])
  • ㅈ + ㅎ = ㅊ (예: 앉 + 히다 → [안치다])

핵심: 소리는 [추카] 하고 하나의 로봇이 되었지만, 글씨는 합체하기 전의 **'기본 로봇'과 '파워 부스터' 두 친구**를 모두 써 줘야 해요! (축하, 입학)

🔍 [합체 로봇 소리를 확인해요! (예시)]

  • 축하 (ㄱ + ㅎ) → 소리는 **[추카]**
  • 입학 (ㅂ + ㅎ) → 소리는 **[이팍]**
  • 좋다 (ㅎ + ㄷ) → 소리는 **[조타]** ('ㅎ'이 앞에 있어요!)
  • 법학 (ㅂ + ㅎ) → 소리는 **[버팍]**
  • 국화 (ㄱ + ㅎ) → 소리는 **[구콰]**

✍️ [나도 할 수 있어! (연습 문제)]

합체! 어떤 로봇으로 변신할까요? 빈칸에 소리 나는 대로 적어보세요.

  1. 급하다 (ㅂ + ㅎ) → [그파다]
  2. 하얗다 (ㅎ + ㄷ) → [하야타]
  3. 밝히다 (ㄺ + ㅎ) → [발키다] (겹받침 중 'ㄱ'이 'ㅎ'과 합체!)

⚠️ [잠깐! '힘센 소리'(4번)와 헷갈리면 안 돼요!]

둘 다 소리가 세지지만, **'ㅎ'의 유무**로 구별해요!

  • 힘센 소리 (4번): [국밥] → [국빱] (합체 X! 'ㅎ'이 없어요!)
  • 거센소리 로봇 (7번): [입학] → [이팍] (합체 O! 'ㅂ'과 'ㅎ'이 합쳐져 'ㅍ'이 되었어요!)

'ㅎ'이 있으면 **7번 거센소리 로봇**이 될 확률이 높아요!

📌 [핵심 요약! (기억 상자)]

[ㄱ, ㄷ, ㅂ, ㅈ] + [ㅎ] → "치익- 합체!" 하고 [ㅋ, ㅌ, ㅍ, ㅊ]으로 변신! 소리는 [추카] (하나)가 되어도, 글씨는 '축하' (둘 다) 쓰기!
